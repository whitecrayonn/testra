# Architecture Documentation

This directory contains Architecture Decision Records (ADRs), diagrams, and runbooks for the Testra platform. The consolidated canonical reference for the current architecture is `../BIBLICAL_TESTRA.md`.

- `../BIBLICAL_TESTRA.md` — Canonical consolidated architecture reference
- `adrs/` — Accepted Architecture Decision Records, including ADR-001 through ADR-012
- `DATABASE_DOCUMENTATION.md` — Storage, migration, and tenancy rules
- `ERD.md` — Entity relationship model
- `SEQUENCE_DIAGRAMS.md` — Request and data sequences
- `SYSTEM_FLOWS.md` — Platform trust and failure flows
- `MODULE_DEPENDENCIES.md` — Modular-monolith dependency rules
- `diagrams/` — Additional system and domain diagrams
- `runbooks/` — Operational playbooks
